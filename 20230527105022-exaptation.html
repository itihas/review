exaptation

Evolutionary mechanism of repurposing - using code, features, or resources already available in the current version of the organism (is there a good word for the thing that evolves?) to try to build new features. This happens because the edit distance between repurposed code and what it repurposes is much shorter, and P(repurposed code) at a timestamp is a function of edit distance and fitness.
