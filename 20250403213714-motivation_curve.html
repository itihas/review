motivation curve

Like a learning curve. Do you have a desire profile around your current task in a project that supports actually doing that task? Does it take "willpower"? Does it afford clarity around what you're getting in return and when? In short, is it paying hedonic rent? And if isn't right now, do you know when it will start to? (And if you don't, why are you doing it? And if you aren't, why are you trying?)
